# Exercise 5 #

Task: Identify the Constructors for each of the following methods and classify each method as either a "Static" or a "Prototype" method:

 * substring
 * create
 * fromCharCode
 * slice
 * toString
 
It might be that more than one constructor provides a method with a given name. Make sure you list them all in your answer. You can limit your search for methods to the String, Object, Array, and Number constructors.


Answer:

substring
Constructor: String
Method Type: Prototype

create
Constructor: Object
Method Type: Static

fromCharCode
Constructor: String
Method Type: Static

slice
Constructor: String, Array
Method Type: Prototype

toString
Constructor: String, Object, Array, Number
Method Type: Prototype
